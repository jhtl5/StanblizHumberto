Análisis de Utilidades en un supermercado

Proyecto final visualización de datos

Por: Jose Humberto Torres L


1. Introdución

En esta actividad se va presentar el análisis de ventas para un supermercado ubicado en Colombia – Departamento de Casanare, el objetivo es presentar las ventas mensuales, categorías de productos más y menos vendidas, utilidades entre otros aspectos de ventas. Todos estos datos corresponden a las ventas en el año 2021, el dataset es propio y también se usó en minería de datos y estadística avanzada.

2. Descripción del Dataset

De acuerdo a lo anterior el Dataset que se utiliza para esta actividad es de ventas, generalmente cuanta con 24 columnas que corresponden a variables y 75414 registros que corresponden a las ventas en el año 2021.

Ejemplo de las variables utilizadas:

  • DIdFactura: Código de la factura asignado de forma automática por el Sistema. Tipo dato Int.
  • IdProducto: Identificación del producto. Tipo dato Int.
  • CodigoAlterno: Código del producto vendido. Tipo de dato Int.
  • DescripcionLarga:Descripción larga del producto vendido. Tipo de dato text.
  • DescripcionCorta:Descripción corta del producto vendido. Tipo de dato text.
  • PrecioCosto:Precio de costo del producto. Tipo de dato Float.
  • PrecioPublicoReal:Precio real para venta al público. Tipo de dato Float.
  • PesoProducto:Peso para el producto. Tipo de dato Float.
  • CantidadS:Total productos vendidos. Tipo dato Float.
  • CantidadE:Total productos que entran. Tipo dato Int
  • IvaVenta:Total IVA aplicado al producto.
  • TotalRegistro: El valor total de la venta de un producto, es diferente a la venta total.

    3.Análisis de Utilidades.

    En esta sesión se hace un análisis de las utilidades de acuerdo a cada una de las categorías de productos, donde evidentemente se observa que los productos que más dejan utilidad son las verduras y desde luego porque es la que más se vende, otro punto importante a tener presente en la categoría de harinas y azucares que también dejan una buena utilidad.

    Con esta información el administrador y dueño del supermercado puede pensar en mejorar la oferta de verduras o bajar un poco el precio de las harinas ya que tiene buena salida y cuando se tienen precios más cómodos atraen más clientes, adicionalmente con esta información pueden lanzar campañas de marketing para mejorar las ventas.     4. Conclusiones

    De acuerdo a este análisis de las utilidades de un pequeño supermercado podemos concluir:

  • Las verduras son los productos que dejan un mejor margen de utilidad con respecto a las otras categorías. Por tanto, es importante crear campañas y planes para fortalecer las ventas y aumentar la utilidad.
  • Los aceites y azucares son productos que tienen buena salida y figuran entre los más vendidos, igualmente es importante generar estrategias para fortalecer las ventas de estos productos.
  • El primer trimestre del año las ventas disminuyen con respecto a los otros trimestres, se debe analizar por qué y tratar de crear campañas para equilibrar las ventas.
